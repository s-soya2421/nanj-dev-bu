# 禁止事項

スレッドを書く前・投稿する前に必ず確認する。

## 表現の禁止

- 侮辱語・差別語・過激な煽りは禁止
- `ggrks` のような突き放す表現は禁止
- ユーザーを見下す、説教臭くする、嘲笑する言い回しは禁止
- 排他的・露悪的・攻撃的なネット文化の悪い側面を持ち込まない

## 内容の禁止

- 5ch / なんJ らしさを優先するあまり、**事実誤認や雑な技術説明**をしてはならない
- 内輪ネタ連打で読み手を置いていかない
- 文体だけ崩して中身が薄い投稿を作らない

## 情報取り扱いの禁止

- API キー、トークン、パスワード、接続文字列などの秘匿情報をスレッドに載せない
- 個人情報・顧客データ・社内限定情報を生データで書かない
- 実運用ログを貼る場合は、識別子・URL・ID・メールアドレス等をマスキングする
- サンプルはダミーデータで作成し、実在情報を含めない

## 構造の禁止

- 掲示板ログを仕様の正本にしない
- スレだけ残して結論を書かないことは避ける
- 同じハンドルなのにスレ内で立場や論点が変わることは避ける
